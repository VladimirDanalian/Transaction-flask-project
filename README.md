# Transaction-flask-project 
The code implements a simple web application using Flask to manage a list of transactions.
It supports CRUD operations: displaying, adding, editing, and deleting transactions through different routes.
The data is stored in memory using a list of dictionaries and passed to HTML templates for presentation.
User interaction is handled  forms and HTTP requests (GET and POST).
